# 🧘‍♀️ TriEase – 3-in-1 Muscle Reliever System

## 📌 Overview
TriEase is an innovative smart healthcare solution designed to provide relief for muscle pain in the **head, wrist, and knee**. It integrates a **mobile application with IoT-based hardware** to deliver a personalized and comfortable massage experience.

This project was developed as part of an academic group project, where I contributed as the **UI/UX designer using Figma & Frontend Developer using Flutter**.

---

## 🎯 Key Features
- 🎛️ Control massage intensity levels
- 🔄 Multiple massage modes
- ⏱️ Timer functionality
- 🎵 Music integration for relaxation
- 🤖 AI-powered personalized assistant (customized bot)
- 🎮 Relaxing mini-games for stress relief
- 📱 User-friendly mobile interface
- ⚡ Real-time control via IoT integration
- 🧵 Adjustable & lightweight wearable design
- 🛡️ Safety features to prevent overheating

## ⚙️ How It Works
1. User wears the TriEase belt
2. Opens the mobile app
3. Selects:
   - Massage type (Head/Wrist/Knee)
   - Intensity level
   - Mode
4. App sends commands to the device via IoT
5. Device provides real-time massage
